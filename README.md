# Système de réservation de bibliothèque

> Projet de cours

## Objectif

Application client-serveur permettant de gérer les réservations 
de livres dans une bibliothèque.

## Architecture

- **client.py** : interface côté utilisateur pour effectuer des réservations
- **server.py** : serveur qui traite les demandes et gère les données
- **src/** : logique métier de l'application
- **tests/** : tests unitaires

## 🛠️ Technologies

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

- Architecture **Client-Serveur**
- Gestion de connexions réseau en Python

## Installation

```bash
pip install -r requirement.txt
python server.py   # Lancer le serveur
python client.py   # Lancer le client
```
